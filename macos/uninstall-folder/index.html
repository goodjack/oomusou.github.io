如何在 macOS 徹底刪除 App 所留下的設定檔 ?

幾個需要手動刪除檔案的目錄
Contents
  1. Version
  2. Directory
  3. Concluson

關於 macOS 的原生 app，雖然可以透過 AppCleaner 或原廠提供了 Uninstall App 移除，但實務上仍會發現一些 app 自己的 設定檔 或 暫存檔，無法百分之百的從系統移除，需手動加以處理。

Version

macOS High Sierra 10.13.3

Directory

這幾個目錄，是 app 常見的 設定檔 與 暫存檔 使用目錄，除了依賴 AppCleaner 外，這幾個目錄最好手動檢查一下，是否有尚未移除檔案。

Concluson

  • 目前在 macOS 尚未有完美的 反安裝 App 解決方案時，手動檢查這幾個目錄是必要的。
